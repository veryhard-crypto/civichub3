📁 FOLDER CONTENTS:
   - This entire folder contains your Civic Hub project
   - Keep all files together and don't delete anything

🚀 QUICK START:

   1. Open Terminal (Cmd + Space, type "Terminal")
   
   2. Navigate here:
      cd ~/path/to/civic-hub-main/civic-hub-main
   
   3. Follow the Mac setup guide:
      Open: SETUP_MAC.md

📚 DOCUMENTATION FILES IN THIS FOLDER:

   ├── SETUP_MAC.md         ⭐ START HERE (Mac-specific setup guide)
   ├── START_HERE.md        (Quick 5-minute guide)
   ├── README.md            (Full project documentation)
   └── README_FOR_MAC_USER.txt  (This file)

🌐 LIVE WEBSITE:
   https://citizen-spark-engine.lovable.app/

🎯 KEY FEATURES:
   ✓ Report issues with GPS location and photos
   ✓ Interactive real-time map
   ✓ Photo upload capability
   ✓ Issue tracking system
   ✓ NGO and citizen portals

💾 DATABASE OPTIONS:
   
   Option 1: MongoDB Atlas (Cloud) - EASIEST
   → Free account at mongodb.com/cloud/atlas
   
   Option 2: Local MongoDB
   → Install via Homebrew: brew install mongodb-community

⚙️ RUN THE APP:

   Terminal 1 (Backend):
   cd server
   npm run dev
   
   Terminal 2 (Frontend):
   npm run dev
   
   Then open: http://localhost:8080
